Customer credit line limit

Description

This module allows to limit the amount of sale orders of a particular customer without invoicing

If you try to create a sale order for a customer that exceeds the unbilled sales limit, the application will show an error to prevent it.

Configuration

  1. Go to Sales Menu >> Orders >> Customers
  2. Click on the customer to limit the sales amount
  3. Go to "Sales & Purchases" tab
  4. Enable "Set Sales Limit" option
  5. Set the limit on "Credit Limit" field
